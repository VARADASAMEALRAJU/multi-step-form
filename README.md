# Multi-Step Registration Form

A functional multi-step registration form built with React, Zustand, and Tailwind CSS.

## Live Demo
**URL:** [https://multi-step-form-lovat-eta.vercel.app/](https://multi-step-form-lovat-eta.vercel.app/)

## Setup Instructions
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the local development server.
4. Open `http://localhost:5173` in your browser.

## Features
- 4-step process with a final Review Summary.
- Real-time validation using Zod and React Hook Form.
- Data persistence via LocalStorage (Zustand).
- Country selection using the REST Countries API.
- Fully responsive and accessible design.
